# Hangman

A console-based Hangman game written in Java. This project was created to practice object-oriented programming and design.

## Features

- Random word selection from a text file
- ASCII hangman display
- Input validation
- Duplicate guess detection
- Win and loss detection

## Technologies

- Java
- IntelliJ IDEA
- Git

## How to Run

### Option 1: IntelliJ IDEA

1. Clone or download the repository.
2. Open the project in IntelliJ IDEA.
3. Make sure `words.txt` is in the project folder.
4. Run `Main.java`.

### Option 2: Command Line

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Change to the project directory:

```bash
cd Hangman
```

3. Compile the program:

```bash
javac Main.java
```

4. Run the program:

```bash
java Main
```

Make sure Java is installed and `words.txt` is in the project folder.
